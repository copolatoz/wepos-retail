WePOS - Retail v.3.42 (Free Version)

Cocok untuk:
Retail/Merchant/Toko, Butik, Salon (semua penjualan berbasis Retail)

Cek Versi Gratis:
www.wepos.id


Cara Instalasi:
1. Install XAMPP
2. copy-paste folder hasil download aplikasi 'wepos-clothing-free' ke xampp/htdocs/
3. run di browser sesuai nama folder yang digunakan, default: http://localhost/wepos-clothing-free


Mengganti URL:
http://localhost/namatokoanda
1. ganti nama folder download 'wepos-clothing-free' menjadi 'namatokoanda'
2. ubah text pada file .htaccess 'wepos-clothing-free' menjadi 'namatokoanda'
*jika .htaccess tidak ditemukan, ubah settingan folder anda agar dapat melihat hidden file dan ekstensi file
