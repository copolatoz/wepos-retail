
Cara Membuat Database Baru
1. buka http://localhost/phpmyadmin
2. creata new database / membuat database baru (dengan nama: 'wepos_retail' sebagai default nama database)
3. pilih database yang baru saja dibuat
4. cari tab / pilihan tombol 'IMPORT' database, pilih file dalam folder wepos/db: 'default-db-retail-v.3.42.21'

Cara Update dari versi sebelumnya:
1. buka http://localhost/phpmyadmin
2. pilih database yang sudah ada / digunakan
3. cari tab / pilihan tombol 'SQ'L database
4. buka file 'update-db-retail-v.3.42.21' dan copy ke 'SQL'
5. jalan kan 'SQL' dengan tombol'Go'
